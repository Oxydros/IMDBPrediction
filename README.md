# IMDBPrediction

Machine Learning project for the course 406 at CSU Long Beach

  - Dataset:
    - https://www.kaggle.com/rounakbanik/the-movies-dataset/data

  - Features:
    - keywords
    - adult
    - budget
    - genres
    - original_language
    - production_companies
    - production_country
    - revenue
    - spoken_languages
    - runtime

  - Label:
    - Rating (0 to 5, but x 10 so 0 to 50)

  - output:
    - IMDB Rating 0 to 50

  Algo:
    - Logistic regression
    - softmax
    - SVM
